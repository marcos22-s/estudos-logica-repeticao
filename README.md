## 📝 Sobre o Projeto
Este projeto foi desenvolvido para estudar e demonstrar o funcionamento dos laços de repetição na linguagem C#. Ele contém exemplos práticos de como automatizar tarefas repetitivas e processar sequências de dados.

## 🚀 Funcionalidades
* **Contagem Automática:** Exemplos práticos com as estruturas `for` e `while`.
* **Gerador de Tabuada:** Cálculo dinâmico baseado na entrada do utilizador.
* **Intervalos Dinâmicos:** Permite definir o início e o fim de uma sequência lógica.
* **Manipulação de ListBox:** Exibição organizada de resultados em tempo de execução.

## 🛠️ Tecnologias
* C# (Windows Forms)
* .NET Framework 4.5
